Creation and Evaluation of a Time Series-Based Dataset for Open-Source Systems

About the Dataset

This website refers to comprehensive software evolution dataset with temporal information of 47 static software metrics of 52 open-source Java systems. This dataset was created for the purpose of supporting the researchers in carrying out empirical studies on software evolution.

The time series extracted to the systems metrics are kept in CSV files. Each CSV file refers to a particular metric from a given system. The lines of the file refer to the system's classes, while the columns refer consist of the system's releases. Therefore, a cell (c,r) in this file has the value of a particuar metric collected for the class c in the release r of a specific system.

Extraction Process

The extraction process of this dataset was performed following a methodology of four steps. The first step determined the selection criterion, i.e., the aspects considered relevant to choose the systems. The second step consisted of making releases of the systems' source code. A release in the context of this dataset consists of a bi-week period. The third step collected the static metrics from the systems' releases, and the fourth step generated the time series.

Software Metrics

The metrics above are the ones considered during the creation of the dataset.

Number of Abstract Methods (abstractMethodsQty)

It computes the number of methods with the abstract modifiers present in a class.

Number of Anonymous Classes (anonymousClassesQty)

It counts the number of anonymous classes present in a class.

Number of Assignments (assignmentsQty)

It computes the number of assignments in a class.

Coupling Between Objects (cbo)

It Counts the number of dependencies a class has.

Quantity of Comparisons (comparisonsQty)

It computes the number of comparisons ("==" and "!=" comparators) in a class.

Quantity of Default Fields (defaultFieldsQty)

It counts the number of default fields in a class.

Quantity of Default Methods (defaultMethodsQty)

It counts the number of default methods in a class.

Depth of Inheritance Tree(dit)

It indicates the level in which a class is positioned in an inheritance hierarchy. The position of a class is the distance between its current position and the root of its inheritance tree.

Fanin (fanin)

It measures the number of classes that reference a particular class.

Fanout (fanout)

It computes the number of other classes referenced by a specific class.

Quantity of Final Fields (finalFieldsQty)

It counts the number of final fields in a class.

Quantity of Final Methods (finalMethodsQty)

It counts the number of final methods in a class.

Number of Inner Classes (innerClassesQty)

It counts the number of inner classes present in a class.

Number of Lambda Expressions (lambdasQty)

It counts the number of lambda expressions present in a class.

Loose Class Cohesion (lcc)

It measures the cohesion of a class via direct and indirect connections between visible classes for the cohesion calculation.

Lack of Cohesion of Methods (lcom)

It measures the cohesion of a class by using the notion of similarity its internal methods. This metric's value is between 0 and 1. The lower the value of LCOM, the more cohesive the class.

Lines of Code (loc)

It counts the lines of count, ignoring empty lines and comments in a class.

Number of Log Statements (logStatementsQty)

It counts the number of log statements in the source code of the classes.

Quantity of Loops(loopQty)

It counts the number of loops (for, while, do while, enhanced for) present in a class.

Quantity of Math Operations (mathOperationsQty)

It measures the number of math operations (times, divide, remainder, plus, minus, left shit, right shift) existing in a class.

Max nested blocks(maxNestedBlocksQty)

It measures the highest number of blocks nested together in a class.

Modifiers (modifiers)

It measures the number of modifiers (public, abstract, private, protected, and native) present in a class.

Number of Children (noc)

It measures the number of immediate subclasses of a particular class in an inheritance hierarchy.

Number of static invocations (nosi)

It counts the number of invocations to static methods.

Quantity of Number (numbersQty)

It counts the number of numbers (int, long, double, float) literals in a class.

Quantity of Parenthesized Expressions (parenthesizedExpsQty)

It measures the number of expressions inside parenthesis.

Quantity of Private Fields (privateFieldsQty)

It counts the number of private fields in a class.

Quantity of Private Methods (privateMethodsQty)

It counts the number of private methods in a class.

Quantity of Protected Fields (protectedFieldsQty)

It counts the number of protected fields in a class.

Quantity of Protected Methods (protectedMethodsQty)

It counts the number of protected methods in a class.

Quantity of Public Fields (publicFieldsQty)

It counts the number of public fields in a class.

Quantity of Public Methods (publicMethodsQty)

It counts the number of public methods in a class.

Quantity of Returns (returnQty)

It measures the number of return instructions.

Response for a Class (rfc)

It counts the number of unique method invocations in a class.

Quantity of Static Fields (staticFieldsQty)

It counts the number of static fields in a class.

Quantity of Static Methods (staticMethodsQty)

It counts the number of static methods in a class.

Quantity of String Literals(stringLiteralsQty)

It computes the number of string literals in a class.

Number of Synchronized Fields(synchronizedFieldsQty)

It counts the number of synchronized fields in a class.

Number of Synchronized Methods (synchronizedMethodsQty)

It counts the number of synchronized methods in a class.

Tight Class Cohesion (tcc)

It measures the cohesion of a class via direct connections between visible methods, two methods or their invocation trees access the same class variable.

Number of Fields (totalFieldsQty)

It counts the number of total fields in a class.

Number of Methods (totalMethodsQty)

It counts the number of total methods in a class.

Quantity of Try/Catches (tryCatchQty)

It counts the number of try/catches in a class.

Number of Unique Words(uniqueWordsQty)

It counts the number of unique words in the source code of a class.

Quantity of Variables (variablesQty)

It measures the number of declared variables in a class.

Weight Method Class (wmc)

It counts the number of branch instructions in a class.
